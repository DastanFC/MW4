## 📡 Sobre el proyecto

Web de noticias no oficial dedicada a **Call of Duty: Modern Warfare 4 (2026)**. Publica noticias, rumores, filtraciones, gameplays y novedades sobre armas, mapas, modos, Warzone y DMZ, con un diseño tipo portal de noticias gaming.

## ✨ Características

| | |
|---|---|
| 📰 | Noticias con portada, extracto y contenido enriquecido (imágenes, YouTube, galerías) |
| ⚡ | **Flashes** tipo "última hora" con vídeo, GIF, mp4 o tuits embebidos |
| 🔴 | Cobertura **en directo** con actualizaciones cronológicas sobre una misma noticia |
| 🏷️ | Sistema de etiquetas: Oficial · Rumor · Review · Mapas · Armas · Modos · Warzone · DMZ · Campaña · Skins... |
| 🔍 | Buscador y filtros por categoría |
| 🔗 | Compartir en Twitter/X y copiar enlace directo a cada noticia |
| 💬 | Comentarios integrados con Disqus |
| 📱 | Diseño responsive, pensado también para móvil |


## ⚠️ Aviso

Este es un proyecto de fans, **no oficial**. No está afiliado ni respaldado por Activision, Infinity Ward ni Call of Duty. Todo el contenido informativo se basa en fuentes públicas.
